# Changelog

## [4.3.0] - 2026-06-21（客制化版本）

### ✨ 新增功能
- **作品集查询** (`recall_drawing` 工具)
  - 支持按关键词搜索历史图片
  - 支持按日期（YYYYMMDD）检索
  - 自动发送匹配的图片（最多3张）
- **LLM 工具调用增强**
  - 更详细的工具描述，提升识别准确率
  - 强制规则：画图请求必须调用工具

### 🔧 优化改进
- **提示词构建优化**
  - 用户描述始终放在提示词模板最前面
  - 更符合 SD 提示词习惯
- **日志输出增强**
  - 增加更多调试日志
  - 清晰显示优化后的提示词
- **作品集保存优化**
  - 文件名包含日期和关键词
  - 索引文件记录完整信息

### 🗑️ 移除功能
- **视觉评价功能** (已禁用)
  - 移除 `enable_vision_review`
  - 移除 `vision_provider_id`
  - 移除 `vision_review_prompt`
  - 移除 `review_send_mode`
  - 移除 `_review_image()` 方法
- **图片 URL 生成**
  - 移除 `_generate_image_url()`
  - 移除 `_build_image_url()`

### 🐛 修复问题
- 修复提示词模板包含多个 `{{positive}}` 时的处理逻辑
- 修复作品集索引文件编码问题
